Architecture

Privacy is not a setting.
It is how the system is built.

There is no toggle to turn off cloud sync. There is no cloud layer. StratoSort runs inference, vector search, and file operations in a single desktop process.

Zero network dependency

After model download, StratoSort never contacts the internet. No telemetry, no analytics. Disconnect your cable and it works the same.

In-process AI inference

LLM inference runs inside the application via node-llama-cpp. No separate server, no API calls, no localhost traffic.

Local vector search

Embeddings generated and stored on-device using Orama. Your semantic index is a local file you own.
